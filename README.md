This repository contains all C++ lab implementations for Data Structures and Algorithms (DSA).
It includes searching algorithms, sorting algorithms, basic data structures, and other lab exercises.

## Currently Included Algorithms
1. Linear Search
2. Binary Search
3. Bubble Sort

## Features
- User input support
- Clean and readable output
- 1-based position output for search operations
- In-place sorting for sorting algorithms
- Simple, student-friendly code structure

## How to Run
1. Open any .cpp file in a C++ compiler (Code::Blocks, Dev-C++, VS Code)
2. Compile and run the program
3. Enter input values as prompted
4. View the output result
